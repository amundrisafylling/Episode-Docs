_available from v8.2.x_

In edit mode, the context menu now contains a new option for "Set sign factor":

Selecting Sales and inverting the sign factor results in the values showing as positive numbers. The calculations that build on Sales also change.

![image](https://github.com/user-attachments/assets/a180838f-4fe4-4a64-a245-fdfcd17e52c8)

![image](https://github.com/user-attachments/assets/ab772ac5-fbea-4408-a783-f348e505b993)


If you change the sign factor to a parent, all child rows will also invert the sign.
If you change the sign factor of a child row, only that row will change.
If you want the parent total to recalculate, you need to tick of "Recalculate totals" as the parent is a total of all child rows:
