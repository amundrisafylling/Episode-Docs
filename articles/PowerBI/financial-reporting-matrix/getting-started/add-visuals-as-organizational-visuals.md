Adding a visual as an organizational visual can be beneficial for many reasons:

1) Control versioning of visuals
2) Add versions that are not yet available in production
3) Add a visual to the format pane for all in your organizational as default



***

# Microsoft documentation on Organizational visuals:
The steps are explained in Microsofts documentation here:
https://learn.microsoft.com/en-us/fabric/admin/organizational-visuals#add-a-visual-from-a-file

***

# How to add an organizational visual, step by step:

*Requirement: You need admin rights in Power BI to do this.

1. Download a visual file. The file format for visuals is .pbiviz. You can find pre-releases here: https://github.com/Profitbase/PowerBI-visuals-FinancialReportingMatrix/releases
2. Got to https://app.powerbi.com/ and sign in.
3. Click the settings symbol in the top right, and then select "Admin portal":


<img width="376" height="813" alt="image" src="https://github.com/user-attachments/assets/86cc49d7-377d-477c-9021-565537a448b8" />

4. Select "Organizational visuals" and click "Add visual", then select "From file":

<img width="1816" height="712" alt="image" src="https://github.com/user-attachments/assets/cec17df8-dd7e-48ed-b003-a1cb08a0bfab" />

5. Browse for the file you downloaded in step 1. Give it a name. Then update an icon if you want. Finally click "Add".

<img width="1838" height="892" alt="image" src="https://github.com/user-attachments/assets/26c4fd37-195e-4166-a15b-cd380c166c0f" />

6. Then, if you select the visual, you can click on "Enable for Vizualization Pane". This will make the organizational visual appear by default in the visualization pane in Power BI for all users in your organization.

<img width="1685" height="408" alt="image" src="https://github.com/user-attachments/assets/87bc92ad-f0f5-4d22-8079-cfec45ad34f1" />


7. If you don't enable it, you can find it in Power BI by clicking "Get more visuals" and selecting the "Organizational visual" tab:

<img width="283" height="470" alt="image" src="https://github.com/user-attachments/assets/517a7744-5ad8-49d5-8114-8ad3c35338ca" />

<img width="799" height="480" alt="image" src="https://github.com/user-attachments/assets/acf6a896-e6d6-4472-9833-63de4b7dc09f" />

8. You will now see two versions side by side. One from AppSource and one from Organizational visual, and you can chose which one you want to use. If you want to change from one visual to the other, you select the visual on your report page, and select the other visual in the format pane:

<img width="890" height="424" alt="image" src="https://github.com/user-attachments/assets/7c925dae-4f16-4334-82b0-68ea2b476444" />

<img width="877" height="395" alt="image" src="https://github.com/user-attachments/assets/07d3ce51-03fa-4c1d-a948-bcf098172ac8" />
