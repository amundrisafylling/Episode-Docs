_available from v8.1.x.x_

Due to stress testing the performance of the visual when certifying the visual, it has a data limit set to 3000 rows.

From v8.1.x.x, we have made this limit available for report developers to adjust as they seem fit, until a global maximum set by Microsoft at 30 000 rows.
The developers must then find a balance between performance, and the amount of data allowed in the visual.

The option is found in the format pane, under "Data limit".
