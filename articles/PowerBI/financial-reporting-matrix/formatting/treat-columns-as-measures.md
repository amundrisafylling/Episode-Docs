## Use case:
Adding conditional formatting is usually done measure by measure.

Using calculation groups, you could add one column header, and one single measure.
Each column could have it's own calculation. This feature will allow users to apply different formatting to these calculations.

With the feature turned OFF, all columns get the same formatting:
![image](https://github.com/user-attachments/assets/b0b49d39-7330-4cc5-a189-365e20a4d266)

With the feature turned ON, the formatting will treat the headers as separate measures, and be applied where selected:
![image](https://github.com/user-attachments/assets/18e1e150-8524-4486-8bec-fd076ccb7a3e)

With a second column header, the formatting will be applied to each column with the same name, as if it was a measure:
![image](https://github.com/user-attachments/assets/abee9a73-f39e-4452-8854-557f6844ba68)
