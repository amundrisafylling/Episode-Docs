## Turn ON/OFF stepped layout

Under "Row Expansion" in the formatting pane, there is an option called "Stepped Layout":

![image](https://github.com/user-attachments/assets/12b60062-81e2-42d4-9795-d0965f5b3fc9)

When ON, children in the row hierarchy will be expanded in a separate column, instead of being showed below the parent.

![image](https://github.com/user-attachments/assets/859c8f70-d03a-4947-a918-0450fd048945)

![image](https://github.com/user-attachments/assets/eb17af53-8fac-441e-949a-c28fffe0080a)

## Parent labels
Turn OFF the "Show parent label" to avoid parent name shown on all rows. Here "Sales" is only visible on the parent row:
